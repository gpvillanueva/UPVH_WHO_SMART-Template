Resource Profile: Imaging Study (IPS)

Official URL: http://hl7.org/fhir/uv/ips/StructureDefinition/ImagingStudy-uv-ips Version: 0.2.1
Active as of 2023-09-25 Computable Name: ImagingStudyUvIps

This profile represents the constraints applied to the ImagingStudy resource by the IPS project, which specifies the information on a DICOM imaging study this imaging result is part of.

Usage:

  • Use this Resource Profile: Bundle - IPS
  • Refer to this Resource Profile: Observation Results: radiology (IPS)
