Requirements

Functional Requirements 1. Ability for participating clinics to access Hepatitis B and C “Case Report” forms online. 2. Ability for participating clinics to access Hepatitis B and C “Care” forms online. 3. Ability for participating clinics to write data to the demonstration FHIR server. 4. Ability for project owners to extract FHIR data from the demonstration server in a format suitable for data visualization or statistical computing tools like “R” for statistical analysis. 5. Ability of participating clinics to read their data, including patient demographics, from the demonstration FHIR server. 6. Ability to limit the access to authenticated users only. Stakeholders 1. Department of Health 1.a Epidemiology Bureau (EB) 1.b Disease Prevention and Control Bureau (DPCB) 1.c Knowledge Management and Information Technology Services (KMITS) 2. Participating Health Facilities / Clinics 2.a Screening, Assessment, and Treatment Facilities (SATF) - Level 1 - 2 Hospitals and Health Centers 2.b End Referral Facility (ERF) - Level 3 Hospitals 3. Specialty Societies - Hepatology Society of the Philippines 4. Epidemiologist User Stories 1. Health Facilities / Clinics 1.a As a member of a participating Health Facility, I want to be able to submit patient Hepatitis B and C data to the demonstration FHIR server through electronic form submission. 1.b As a member of a participating Health Facility, I want to be able to read electronic Hepatitis B and C data from the demonstration FHIR server. 1.c As a member of a participating Health Facility, I want to be able to edit Hepatitis B and C data in the demonstration FHIR server. 2. Epidemiologist 2.a As a field epidemiologist, I want to be able to read individual patient Hepatitis B and C data from the demonstration FHIR server. for track and trace purposes. 2.b As a research epidemiologist, I want to be able to electronically access anonymized collected form data so that I can do statistical analysis.
