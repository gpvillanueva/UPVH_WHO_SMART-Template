Data Type Profile: Quantity (IPS)

Official URL: http://hl7.org/fhir/uv/ips/StructureDefinition/Quantity-uv-ips Version: 0.2.1
Active as of 2023-09-25 Computable Name: QuantityIPS

Data type Quantity constrained to use UCUM as the code system for units

This profile of the Quantity data type imposes the usage of the UCUM as the code system for units

Usage:

  • Use this DataType Constraint: Observation - SH: alcohol use, Observation Results: laboratory (IPS), Observation Results: pathology (IPS), Observation Results: radiology (IPS) and Ratio (IPS)
